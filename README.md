# 🌷☕ Bloom & Brew

**Sip • Bloom • Unwind**

Bloom & Brew is a modern and aesthetically designed flower café website that combines the warmth of handcrafted coffee with the beauty of fresh floral arrangements. Inspired by cozy cafés and Pinterest-style interiors, the project provides an elegant digital experience with interactive sections and responsive design.

---

## ✨ Overview

Bloom & Brew is a single-page responsive website built using **HTML, CSS, and JavaScript**. The website showcases a fictional flower café where customers can enjoy premium coffee, delicious desserts, and beautiful bouquets in a relaxing atmosphere.

This project focuses on creating a visually appealing user experience while demonstrating frontend development skills.
