# 🌷☕ Bloom & Brew

**Sip • Bloom • Unwind**

Bloom & Brew is a modern, aesthetically designed flower café website that blends the warmth of handcrafted coffee with the beauty of fresh floral arrangements. Inspired by cozy cafés and Pinterest-style interiors, this single-page frontend project delivers an elegant digital experience with interactive sections, responsive layout, and smooth user interactions.

---

## 📌 About the Project

Bloom & Brew is a fictional flower café website built using **HTML, CSS, and Vanilla JavaScript**. It showcases a stylish café concept where visitors can explore premium coffee, delicious desserts, beautiful bouquet collections, and a relaxing ambience.

The main goal of this project is to practice modern frontend development skills while creating a visually pleasing, mobile-friendly, and interactive website.

---

This project is ideal for:
